文档

__init__(self str ext = "")

    参数 ext: 文件扩展名。
    点击open按钮可打开QFileDialog组件来选择文件。
    空字符串:   QFileDialog可查看选择文件夹;
    *：         QFileDialog可查看选择任意扩展名的文件;
    str:        QFileDialog可查看选择指定扩展名的文件;
    list[str,]：QFileDialog可查看选择列表内所有扩展名的文件;

str __repr__(self)

    返回 Path(ext)。

str ext

    文件扩展名。

str arg

    str arg = Path.arg: 查询文件选择框文本。
    Path.arg = str arg: 设置文件选择框文本。

void argChanged(str)

    当文件选择框文本改变时，发射文件选择框文本。

案例

import QFn.args


def show_arg(arg):
    print "arg changed:", arg

widget = QFn.args.Path()
widget.show()

print widget
# Result: Path()

new_widget = eval(repr(widget), QFn.args.__dict__)
new_widget.show()

new_widget.argChanged.connect(show_arg)

new_widget.arg = "C:/Users/Admin/Documents"
print new_widget.arg
new_widget.arg = 0
# Warning: Path().arg must be str
