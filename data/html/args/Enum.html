QFn.args.Enum

父类：

  • PySide.QtGui.QComboBox

方法：

  • __init__(self, *texts)
  • str __repr__(self)
  • paintEvent(self, QPaintEvent event)

属性：

  • list texts
  • str arg

信号：

  • void argChanged (str)

文档

__init__(self, *texts)

    参数 texts: 每下拉菜单文本(str,str……)
    为texts中每个文本添加下拉菜单；
    令argChanged = currentIndexChanged[str]

str __repr__(self)

    返回 Enum(*texts)。

paintEvent(self, paintEvent event)

    宽度实时更新为下拉菜单所选项文本所占宽度。

list texts

    每勾选框文本list[str, str……]

str arg

    str arg = Enum.arg: 查询下拉菜单所选项文本。
    Enum.arg = str arg: 设置下拉菜单所选项文本。

void argChanged(str)

    当下拉菜单所选项改变时，发射下拉菜单所选项文本。

案例

import QFn.args


def show_arg(arg):
    print "arg changed:", arg

widget = QFn.args.Enum("enum1", "enum2")
widget.show()

print widget
# Result: Enum('enum1', 'enum2')

new_widget = eval(repr(widget), QFn.args.__dict__)
new_widget.show()

new_widget.argChanged.connect(show_arg)

new_widget.arg = "enum2"
print new_widget.arg
new_widget.arg = "enum3"
# Warning: Enum('enum1', 'enum2').arg must be in ('enum1', 'enum2')
