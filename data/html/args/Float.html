文档

__init__(self, int min_value=-2147483648, int max_value=2147483647)

    参数 min_value: 组件可输入最小值，该值不小于-2147483648。
    参数 max_value: 组件可输入最大值，该值不大于2147483647。
    参数令argChanged = valueChanged。

str __repr__(self)

    返回 Float(min_value， max_value)。

paintEvent(self, paintEvent event)

    浮点数输入框默认为四位数宽度。
    当输入数值大于四位数时，宽度为输入数值所占宽度。

arg

    int Int.arg: 查询整数输入框输入数值。
    Int.arg = int: 设置整数输入框输入数值。

void argChanged(int)

当浮点数输入框数值改变时，发射整数输入框数值。

案例

import QFn.args


def show_arg(arg):
    print "arg changed:", arg

widget = QFn.args.Float()
widget.show()

print widget
# Result: Float()

new_widget = eval(repr(widget), QFn.args.__dict__)
new_widget.show()

new_widget.argChanged.connect(show_arg)

new_widget.arg = 100000.0
print new_widget.arg
new_widget.arg = "10000"
# Warning: Float().arg must be float
