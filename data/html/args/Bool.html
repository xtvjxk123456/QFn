文档

__init__(self)

    令argChanged = toggled。

str __repr__(self)

    返回 Bool()。

bool arg

    bool arg = Bool.arg: 查询勾选框是否勾选。
    Bool.arg = bool arg: 设置勾选框的勾选状态。

void argChanged(bool)

    当勾选框勾选状态改变时，发射勾选框勾选状态。

案例

import QFn.args


def show_arg(arg):
    print "arg changed:", arg

widget = QFn.args.Bool()
widget.show()

print widget
# Result: Bool()

new_widget = eval(repr(widget), QFn.args.__dict__)
new_widget.show()

new_widget.argChanged.connect(show_arg)

new_widget.arg = True
print new_widget.arg
new_widget.arg = 1
# Warning: Bool().arg must be bool
