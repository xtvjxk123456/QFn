文档

__init__(self)

    令argChanged = textChanged。

str __repr__(self)

    返回 String()。

bool arg

    str arg = String.arg: 查询文本输入框的输入值。
    String.arg = str arg: 设置文本输入框的输入值。

void argChanged(str)

    当文本输入框数值改变时，发射文本输入框输入数值。

案例

import QFn.args


def show_arg(arg):
    print "arg changed:", arg

widget = QFn.args.String()
widget.show()

print widget
# Result: String()

new_widget = eval(repr(widget), QFn.args.__dict__)
new_widget.show()

new_widget.argChanged.connect(show_arg)

new_widget.arg = "string"
print new_widget.arg
new_widget.arg = True
# Warning: String().arg must be str
