文档

_sre.SRE_Pattern pattern

    当将文件拖入路径列表框时，将由pattern作为文件命名规则对拖入文件进行筛选。

list paths

    路径列表框记录的路径列表list[str, str……]。

PySide.QtGui.QMenu menu

    点击后执行clear方法。

str arg

    list[str, str……] arg = Path.arg: 查询路径列表框记录的路径列表。
    Path.arg = list[str, str……] arg: 设置路径列表框记录的路径列表。

__init__(self str pattern="")

    参数 pattern: 正则表达式

str __repr__(self)

    返回 PathArray(pattern)。

contextMenuEvent(self, event)

    右击时，在鼠标位置显示menu。

dragEnterEvent(self, event)

    允许携带路径信息的拖拽对象QDrag触发拖拽事件dropEvent。

dropEvent(self, event)

    添加拖拽对象QDrag携带的路径到路径列表框。

clear(self)

    清空路径列表。

void argChanged(str)

    当路径列表框记录的路径列表改变时，发射路径列表框记录的路径列表。

案例

import QFn.args


def show_arg(arg):
    print "arg changed:", arg

widget = QFn.args.PathArray()
widget.show()

print widget
# Result: PathArray()

new_widget = eval(repr(widget), QFn.args.__dict__)
new_widget.show()

new_widget.argChanged.connect(show_arg)

new_widget.arg = 0
# Warning: PathArray().arg must be [str,str……]

new_widget.arg = ["C:/Users/Admin/Documents"]
print new_widget.arg
