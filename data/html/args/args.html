QFn.args

模块简介：

每种组件对应一种参数类型。
组件均继承自QtGui.QWidget。
组件均可通过arg属性来查询/更改组件代表的参数的值。
组件都具有argChang信号，在组件数值改变时发射arg。
组件均可通过__repr__方法与eval函数来与字符串之间进行转换。

类：

类名 参数类型 样式
Bool Bool
BoolArray list[bool, bool……]
Int Int
IntArray list[int, int……]
Float float
FloatArray list[foat, float……]
String str/unicode
Enum str/unicode
Path str/unicode
PathArray list[str, str……]
