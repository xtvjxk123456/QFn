QFn.args.BoolArray

父类：

  • PySide.QtGui.QWidget

方法：

  • __init__(self, str *labels)
  • str __repr__(self)

属性：

  • arg
  • labels

信号：

  • void argChanged (list)

文档

__init__(self, str *labels)

    参数 labels: 每勾选框标签(str,str……)
    依次为labels中每个标签创建相应QCheckBox并横向排列；

str __repr__(self)

    返回 BoolArray(*labels)。

arg

    list[bool, bool……] BoolArray.arg: 查询每勾选框是否勾选。
    BoolArray.arg = list[bool, bool……]: 设置每勾选框的勾选状态。

labels

    每勾选框标签list[str, str……]

void argChanged(list)

    当勾选框勾选状态改变时，发射每勾选框勾选状态。

案例

import QFn.args


def show_arg(arg):
    print "arg changed:", arg

widget = QFn.args.BoolArray("bool1", "bool2")
widget.show()

print widget
# Result: BoolArray('bool1', 'bool2')

new_widget = eval(repr(widget), QFn.args.__dict__)
new_widget.show()

new_widget.argChanged.connect(show_arg)

new_widget.arg = [True, False]
print new_widget.arg
new_widget.arg = 1
# Warning: BoolArray('bool1', 'bool2').arg must be 2 bool list
