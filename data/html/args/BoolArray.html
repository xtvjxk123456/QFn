QFn.args.BoolArray

父类：

  • PySide.QtGui.QWidget

属性：

  • list texts
  • list arg

方法：

  • __init__(self, str *labels)
  • str __repr__(self)

信号：

  • void argChanged (list)

文档

__init__(self, str *texts)

    参数 texts: 每勾选框文本(str,str……)
    依次为texts中每个文本创建相应QCheckBox并横向排列；

list arg

    list[bool, bool……] arg = BoolArray.arg: 查询每勾选框是否勾选。
    BoolArray.arg = list[bool, bool……] arg: 设置每勾选框的勾选状态。

str __repr__(self)

    返回 BoolArray(*texts)。

list texts

    每勾选框文本list[str, str……]

void argChanged(list)

    每当勾选框勾选状态改变时，发射每勾选框勾选状态。

案例

import QFn.args


def show_arg(arg):
    print "arg changed:", arg

widget = QFn.args.BoolArray("bool1", "bool2")
widget.show()

print widget
# Result: BoolArray('bool1', 'bool2')

new_widget = eval(repr(widget), QFn.args.__dict__)
new_widget.show()

new_widget.argChanged.connect(show_arg)

new_widget.arg = [True, False]
print new_widget.arg
new_widget.arg = 1
# Warning: BoolArray('bool1', 'bool2').arg must be 2 bool list
