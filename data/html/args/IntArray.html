QFn.args.IntArray

父类：

  • PySide.QtGui.QWidget

属性：

  • list labels
  • list arg

方法：

  • __init__(self, list labels, int min_value=-2147483648, int max_value=2147483647)
  • str __repr__(self)

信号：

  • void argChanged (list)

文档

__init__(self, list labels, int min_value=-2147483648, int max_value=2147483647)

    参数 labels: 每勾选框标签(str,str……)
    参数 min_value: 组件可输入最小值，该值不小于-2147483648。
    参数 max_value: 组件可输入最大值，该值不大于2147483647。
    依次为labels中每个标签创建相应Int并横向排列。

list labels

    每整数输入框标签list[str, str……]

list arg

    [int, int……] IntArray.arg: 查询每整数输入框输入数值。
    IntArray.arg = [int, int……]: 设置每整数输入框输入数值。

str __repr__(self)

    返回 IntArray(labels, min_value， max_value)。

void argChanged(list)

    每当整数输入框数值改变时，发射每整数输入框数值。

案例

import QFn.args


def show_arg(arg):
    print "arg changed:", arg

widget = QFn.args.IntArray(["x", "y", "z"])
widget.show()

print widget
# Result: IntArray(['x', 'y', 'z'])

new_widget = eval(repr(widget), QFn.args.__dict__)
new_widget.show()

new_widget.argChanged.connect(show_arg)

new_widget.arg = [1, 1, 1]
print new_widget.arg
new_widget.arg = "10000"
# Warning: IntArray(['x', 'y', 'z']).arg must be 3 int
