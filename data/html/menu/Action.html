文档

str path

    功能菜单对应文件路径。

Window window

    功能函数窗口。
    窗口是通过读取对应yaml文件创建的。
    窗口在菜单第一次点击时才会创建。

__init__(self, str path, Menu parent=None)

    参数 path: 功能菜单对应文件路径。
    参数 parent: 父级菜单。
    path是一个去除后缀的相对路径。
    例：D:/menu/test
    需要时再加上对应后缀（.jpg、.yaml……）来对对实际的文件路径进行查询使用。

read(self)

    点击菜单时调用的函数。
    读取yaml,若yaml中仅记录一个无参函数，则直接执行。
    否则，创建并显示window。

write(self, Window window)

    参数 window: 功能函数窗口。
    将window窗口写入对应yaml文件。
