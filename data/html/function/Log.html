QFn.function.Log

  • object

  • str error

方法：

  • __init__(self, str path)

文档

str path

    日志路径。

str time

    当前时间。
    例：2017/09/06 22:48

str error

    try捕捉到的错误信息。

__init__(self, str path)

    参数：path 日志路径。

str __repr__(self)

    返回 QFn.function.Log(path)。

__call__(self, str *texts)

    参数：texts需要记录到日志的文本。
    插入texts到日志开头并打印texts。
    超出一千行的部分自动丢弃。

open(self)

    打开日志供用户观看。

案例

import QFn.function

log = QFn.function.Log("D:/QFn/test.log")
print log
print log.path
print log.time

try:
    1/0
except ZeroDivisionError:
    log("又偷偷写BUG", log.time, log.error)
    log.open()
