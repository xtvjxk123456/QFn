文档

PySide.QtGui.QMessageBox message

    仅在首次调用该属性时创建窗口实例，避免在导入模块时创建窗口实例而出现问题。

__init__(str title)

    参数 title: 窗口标题，用来区别信息类型。

__call__(str text)

    参数 text: 提示内容。同一种信息类型共用一种提示窗口实例。
    将text内容在提示窗口中显示来对用户进行提示。

案例

import QFn.window

message = QFn.window.Message("test info")
message(
    u"""
    测试用
    一一一一一一一一一一一一一一
    二二二二二二二二二二二二二二二二二二
    """
)
