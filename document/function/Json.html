文档

str path

    json文件路径。

bool __init__(self, str path)

    参数 path：json文件路径。

bool __repr__(self)

    返回：QFn.function.Json(path)

bool __nonzero__(self)

    返回：json文件是否存在。

str read(self)

    返回：json文件记录的数据。

str reload(self)

    当json文件以write函数之外的方式发生改变时，用reload来更新数据。
    返回: json文件记录的Python数据。

write(self, object data)

    参数 data: python数据。
    将python数据写入json文件。

delete(self)

    删除json文件。
