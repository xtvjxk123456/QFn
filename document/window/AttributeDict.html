文档

object __getattr__(str attr)

    参数 attr: 属性名。
    返回: 将attr当作字典中的key查询到的值。
    当通过AttributeDict.attr访问dict不存在的属性时调用。

案例

import QFn.window

mayaFile = QFn.window.AttributeDict(ma=0, mb=1)
print mayaFile
print mayaFile.ma
print mayaFile["mb"]
