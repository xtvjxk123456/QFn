QFn.enum

说明

    只要加载了QFn, enum就会添加到内建模块内。
    enum可以与字符串和整数进行对比转化。
    enum可以通过__getitem__与__getattr__访问各枚举项。

案例

import QFn
SEX = enum("man", "woman")
print repr(SEX)
#result enum('man', 'woman)
print SEX.man
#result man
print SEX.woman
#result woman
print SEX[0]
#result man
print SEX[1]
#result woman
print int(SEX.woman)
#result 1
print str(SEX[0])
#result man
print SEX.man == 0
#result True
print SEX[1] == "woman"
#result True
